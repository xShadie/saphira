[CENTER_ALIGN;FONT_SIZE=18;COLOR=58ADFF]FUNKTIONALITÄT DER VERSCHIEDENEN ABSCHNITTE

[X_PADDING=12]1. Mein Shop: In diesem Bereich können Sie Ihren Offline-Shop erstellen, rechts oben finden Sie eine Vorschau des
[X_PADDING=12]Shop, den Sie verwenden werden, durch Klicken auf die Pfeile können Sie auch die speziellen Shops auswählen, aber seien Sie vorsichtig,
[X_PADDING=12]Um die Spezialgeschäfte zu nutzen, benötigen Sie 1 Kaschmirbündel.

[X_PADDING=12]2. Store-Liste: In diesem Abschnitt können Sie alle aktiven Offline-Stores anzeigen.

[X_PADDING=12]3. Lager: In diesem Abschnitt finden Sie alle Artikel, die Sie nicht verkauft haben, und die Einnahmen aus den Verkäufen.

[X_PADDING=12]4. Le mie offerte: in questa sezione sono elencate tutte le offerte che hai effetuato.

[X_PADDING=12]5. Suche: In diesem Bereich können Sie mit den verschiedenen Filtern gezielt suchen.

[X_PADDING=12]6. Meine Suchen: In diesem Abschnitt sehen Sie alle von Ihnen durchgeführten Suchen.

[X_PADDING=12]7. Gespeicherte Suchen: In diesem Abschnitt werden alle gespeicherten Suchfilter aufgelistet.

[X_PADDING=12]8. Meine Auktionen: In diesem Abschnitt können Sie eine Auktion erstellen oder den Status einer Ihrer laufenden Auktionen überprüfen.

[X_PADDING=12]9. Laufende Auktionen: In diesem Abschnitt werden alle noch laufenden Auktionen aufgelistet.
